# Chessboard

Self-contained 3D chessboard (`index.html`) with a verified full-rules engine.
Open `index.html` directly in a browser, or serve locally:

    npx serve .

For the photorealism upgrade pass, read `CLAUDE.md` and start a Claude Code
session in this directory:

    export REPLICATE_API_TOKEN=r8_...
    claude

Asset generation scaffold: `scripts/generate_assets.mjs` (see header comments).
